# Telas do App

## Militar
1) Login (matrícula + senha)
2) Vagas disponíveis (lista/cartões)
- Mostra: ALA, Data, Serviço, Sobreaviso
- Botões: Selecionar / Remover / Indisponível
3) Minhas escolhas (lista e status)

## Administrador
1) Cadastro de militares e licenças
2) Cadastro/importação da escala do mês
3) Regras de sobreaviso e cobertura
4) Relatório + Exportar Excel
